W momencie odpalenia pliku .apk, jeżeli użytkwonik nie przyznał uprawnień do instalowania aplikacji z nieznanych źródeł, 
należy je przyznać, tak jak na filmie. Następnie należy kliknąć przycisk Zainstaluj a na oknie o wykonaniu skanu aplikacji 
kliknąć przycisk Skanuj aplikację lub rozwinąć dialog i zainstalować bez skanowania.
